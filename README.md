# elementary L10n

Translation status viewer for elementary OS components.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install elementary-l10n
```

### Fedora/RPM
```bash
sudo dnf install elementary-l10n
```
